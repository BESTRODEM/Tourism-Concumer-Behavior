🔥 Firebase 연결 설정

데이터를 클라우드에 저장하고 집필진이 함께 접속하려면
Firebase 프로젝트를 연결해야 합니다. 최초 1회만 설정합니다.

1
console.firebase.google.com 접속 → 프로젝트 만들기
2
프로젝트 → Firestore Database 메뉴 → 데이터베이스 만들기 (테스트 모드)
3
Storage 메뉴 → 시작하기 (테스트 모드)
4
프로젝트 설정 → 내 앱 → 웹 앱 추가 → SDK 설정 코드 복사
5
아래 입력란에 firebaseConfig = { ... } 부분의 JSON을 붙여넣고 연결
설정 코드가 올바르지 않습니다. JSON 형식을 확인하세요.

📐 단원 구성 안내

이 툴은 최대 4단계의 단원 계층 구조를 지원합니다.
🟦 대단원 관리자 전용
책의 최상위 구조. 관리자만 생성·수정·삭제 가능. 내용 직접 입력 없이 하위 단원을 담는 컨테이너.
예) 제1부. 소비자행동의 기초
🟩 중단원 담당 집필진
대단원 아래 장(章) 단위. 배정된 집필진이 내용 작성·상태 변경 가능. 하위 단원도 직접 추가 가능.
예) 제1장. 소비자행동의 개념과 의의
🟨 소단원 담당 집필진
중단원 내 절(節) 단위. 해당 중단원 담당 집필진이 추가·수정·삭제 가능.
예) 제1절. 소비자행동의 정의
🟪 소소단원 담당 집필진
가장 세분화된 항(項) 단위. 소단원 담당 집필진이 추가·수정·삭제 가능.
예) 1. 소비자행동의 학문적 배경
⚠️ 권한 요약: 대단원 → 관리자 전용 / 중·소·소소단원 → 배정된 집필진만 집필 가능, 미배정·다른 집필자는 🔒 잠금 표시
